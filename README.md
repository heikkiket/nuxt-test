# MANUAL

1. Clone the repository

2. Test demo site for example with

    cd nuxt-test/dist
    python -m SimpleHTTPServer

3. Navigate to http://localhost:8000


## Development


 ( sudo apt install nodejs npm)

    npm install
    
    npm run dev
    
    
    (dev site runs in localhost:3000)


Compiling the site:

    npm run generate
